# First_repo

This repository now includes a small Python game that demonstrates advanced
techniques such as object-oriented design and the minimax search algorithm.

## Tic-Tac-Toe with AI

The `tictactoe.py` module provides an unbeatable AI opponent. You can play
against the computer or watch two AI players compete.

### Requirements

- Python 3.8 or later (no external dependencies)

### Usage

Play against the AI:

```bash
python tictactoe.py
```

Watch an automated demo where two AI players face off:

```bash
python tictactoe.py --demo
```
